### Enhanced RouterHost with Anchor Navigation Support

This version introduces comprehensive anchor navigation support to the RouterHost:

- **Hash Anchor Support** - Links with hash fragments (e.g., `#section-id`) now work correctly for same-page navigation
- **Cross-Route Anchors** - Navigate to different routes with anchor targets (e.g., `/about#team`) and automatically scroll to the target element
- **Smart Scroll Behavior** - Automatic smooth scrolling to anchor elements after navigation
- **Scroll to Top** - Pages automatically scroll to top when navigating without hash anchors
- **Query Parameter Preservation** - URL query parameters are now preserved during navigation

### Improved Navigation Experience

- **Same-Page Anchors** - Hash-only links on the current page use native browser scrolling behavior for better performance
- **Smooth Scrolling** - Cross-route anchor navigation includes smooth scroll transitions to target elements
